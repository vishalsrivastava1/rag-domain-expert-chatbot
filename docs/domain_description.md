# Domain Description

Domain: NASA space research and technical publications.

We collected 60 publicly available NASA PDF documents including mission reports,
technical publications, scientific reports, annual reports, and space exploration materials.

The raw files are stored in `data/raw/`, and document source information is recorded
in `docs/source_list.csv`.

This dataset will be used as the knowledge base for a production-ready RAG chatbot
that answers questions about NASA missions, aerospace technology, and space science.
